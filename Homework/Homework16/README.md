# Homework 16: AWS Practice

## Опис завдання

У цій домашній роботі було виконано практичну роботу з основними сервісами AWS:

- EC2
- S3
- IAM

Основна мета роботи — навчитися створювати, перевіряти та видаляти AWS-ресурси, а також налаштовувати обмежений доступ користувача до конкретного S3 bucket.
